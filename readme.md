# Hello World

Says hello fellow rustaceans and outputs an ASCII drawing of a crab.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install rust
2. Open CMD, navigate to the project directory
3. Run command "cargo run". Program will output to console.

Instructions for using the software:

1. In a terminal, navigate to the project directory
2. Run command "cargo run"
3. Admire the crab

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Latest version of Rust
